## Testing Instructions

### To Test Report Generation:
1. Open the Parksons Maintenance System spreadsheet
2. Go to menu: Maintenance System → Test Report Generation
3. Review the test results in the alert dialog
4. Check the Logger for detailed test output

### To Use Report Generation:
```javascript
// Build a report
var config = {
  title: 'Monthly Report',
  metrics: ['MTTR', 'MTBF', 'Availability'],
  dimensions: ['Machine', 'Shift', 'Category'],
  filters: { fy: 'FY 2024-25' },
  visualizations: ['Summary', 'Charts']
};
var report = buildReport(config);

// Export to PDF
var pdfResult = exportReportPDF(report);

// Export to Excel
var excelResult = exportReportExcel(report);

// Save as template
saveReportTemplate('Monthly Report Template', config);

// Schedule report
scheduleReport(config, 'monthly', ['manager@company.com']);
```
